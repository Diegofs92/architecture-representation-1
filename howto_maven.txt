Este projeto usa maven (http://maven.apache.org) para gerenciamento de dependências. A maioria dos jars foram encontrados na internet e estão 
configurados no pom.xml. Entretanto, algumas versões não foram encontradas 

Para isso você  precisa baixar esses jars e instalar manualmente em seu repositório. Para isso acesse o seguinte link:

https://dl.dropbox.com/u/6730822/libs/libsArquitetura.zip

Caso use linha de comando:

curl -O https://dl.dropbox.com/u/6730822/libs/libsArquitetura.zip
unzip libMestrado.zip

Após ter os jars baixados rodar os seguintes comandos:


mvn install:install-file -Dfile=/path/to/org.eclipse.uml2.uml.resources_4.0.0.v20120604-0919.jar -DgroupId=org.uml2.resources4 -DartifactId=uml2-resources -Dversion=4.0.0 -Dpackaging=jar

mvn install:install-file -Dfile=path/to/org.eclipse.uml2.uml.profile.l3_1.0.0.v20120604-0919.jar -DgroupId=org.uml2.profile.l3 -DartifactId=uml2-profilel3 -Dversion=1.0.0 -Dpackaging=jar

mvn install:install-file -Dfile=/path/to/org.eclipse.uml2.uml.profile.l2_1.0.0.v20120604-0919.jar -DgroupId=org.uml2.profile.l2 -DartifactId=uml2-profilel2 -Dversion=1.0.0 -Dpackaging=jar

mvn install:install-file -Dfile=path/to/org.eclipse.uml2.types_1.0.0.v20120604-0919.jar -DgroupId=org.uml2.types -DartifactId=uml2-types -Dversion=1.0.0 -Dpackaging=jar

mvn install:install-file -Dfile=/path/to/org.eclipse.uml2.common_1.7.0.v20120604-0919.jar -DgroupId=org.uml2.common -DartifactId=uml2-common -Dversion=1.7.0 -Dpackaging=jar

mvn install:install-file -Dfile=/path/to/org.eclipse.uml2.uml_4.0.0.v20120604-0919.jar -DgroupId=org.uml2.uml24 -DartifactId=uml2-uml24 -Dversion=4.0.0 -Dpackaging=jar

Após isso rodar:

mvn install
